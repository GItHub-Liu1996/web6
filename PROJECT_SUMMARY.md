## 🛠️ 开发指南

### 启动项目
```bash
# 安装依赖
npm install --legacy-peer-deps

# 启动开发服务器
npm run dev

# 访问网站
open http://localhost:3000
```

### 开发命令
```bash
# 代码检查
npm run lint

# 自动修复
npm run lint:fix

# 类型检查
npm run type-check

# 运行测试
npm run test

# 构建生产版本
npm run build
```

### 开发规范
- 所有组件必须包含 JSDoc 注释
- 使用 TypeScript 严格模式
- 遵循单一职责原则
- 优先使用组合而非配置
